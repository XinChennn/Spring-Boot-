# 基于Spring Boot的教务管理系统、学生管理系统、课表查询系统  

## 一、系统介绍

本作品主要实现了一个课表查询系统，采用了SSM（Spring + SpringMVC + MyBatis）的基础架构。

## 二、使用技术

1. spring-boot
2. spring-MVC
3. thymeleaf
4. mybatis-plus
5. druid
6. Lombok
7. Hutool
8. H-ui
9. Timetables.js

## 三、项目具体介绍

系统能够实现用户的登录退出，教室、班级、院系、地点、教工、学生、用户、课程、课程表的分页展示、增加、删除（批量）、修改、根据条件查询，排课（课程表增删改查）、课程表通过表格展示（可以通过教师、班级、学年、学期进行筛选）。
